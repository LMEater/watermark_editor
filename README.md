# 图片水印编辑器

一个简单易用的图片水印编辑工具，支持粘贴图片、编辑水印、导出处理后的图片。

## 功能特点

- 支持剪贴板直接粘贴图片
- 水印文字支持多行输入
- 可调整水印字体大小、颜色和透明度
- 支持多个水印位置选项
- 可以直接复制或下载处理后的图片

## 使用方法

1. 双击运行 `watermark_editor.exe`
2. 在浏览器中自动打开程序界面
3. 使用 Ctrl+V 粘贴图片
4. 编辑水印文字和样式
5. 点击"应用水印"预览效果
6. 使用"复制图片"或"下载图片"保存结果

## 注意事项

- 程序运行时会在系统托盘显示图标
- 关闭浏览器后，可以右键点击托盘图标选择退出程序
- 首次运行可能需要允许通过防火墙
